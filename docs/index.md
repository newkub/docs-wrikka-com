## Welcome to Wrikka Framework

Wrikka is a comprehensive development framework designed for modern web applications. It provides everything you need to build fast, scalable, and maintainable applications.

## Quick Start

```bash
# Install the CLI
bun add -g @wrikka/cli

# Create a new project
wrikka create my-app

# Start development
cd my-app
bun dev
```
