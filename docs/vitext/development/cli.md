# Vitext CLI Commands

| คำสั่ง | คำอธิบาย |
|--------|-----------|
| `vitext add <feature>` | เพิ่มฟีเจอร์ต่างๆ ให้โปรเจ็กต์ (auth, i18n, testing) |
| `vitext build` | build สำหรับ production |
| `vitext cleanup` | ลบไฟล์ที่ไม่จำเป็น |
| `vitext create` | สร้างโปรเจ็กต์ใหม่ |
| `vitext dev` | เริ่ม development server |
| `vitext generate` | generate code/components |
| `vitext info` | แสดงข้อมูลเกี่ยวกับโปรเจ็กต์ |
| `vitext preview` | preview production build |
| `vitext upgrade` | อัปเกรดแพ็กเกจและ dependencies |
| `vitext version` | แสดงเวอร์ชันของ Vitext |
