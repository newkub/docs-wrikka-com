## Quick Start

```bash
# Install Vitext
bun add @wrikka/vitext

# Launch the editor
bun vitext
```

## Configuration

Vitext uses a simple configuration file (`vitext.config.js`) to customize your experience:

```js
export default {
  theme: "dark",
  fontSize: 14,
  tabSize: 2,
  wordWrap: true,
  minimap: true,
  autoComplete: true,
  aiSuggestions: true,
};
```

## Extensions

Extend Vitext functionality with powerful extensions:
