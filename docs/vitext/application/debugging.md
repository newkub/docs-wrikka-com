## Debugging in Vite + React

### Browser DevTools
- Use Chrome DevTools or Firefox DevTools
* `console.log()` for simple debugging
* `console.table()` for structured data

### React Developer Tools
- Install React DevTools extension
* Inspect component hierarchy
* Check component props/state

### Vite Specific Tools
- Use `--debug` flag in dev server
* `vite-plugin-inspect` for bundle analysis

### Common Issues
1. HMR not working
2. Environment variables undefined
3. Hot reload inconsistent
