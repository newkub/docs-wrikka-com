# Dotfiles Manager

Tools and best practices for managing development configurations

## Features

- Cross-platform configuration management
- Version control for dotfiles
- Synchronization across machines
- Template system for environment-specific settings

## Recommended Tools

```bash
bun add -g @wrikka/dotfiles-manager
```

## Basic Usage

```bash
dm init      # Initialize new dotfiles repo
dm sync      # Sync configurations
dm backup    # Create backup
```
